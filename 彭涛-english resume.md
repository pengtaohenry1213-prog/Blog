# Frontend Engineer

## Responsibilities:
* Develop strong frontend applications that meet project specifications and deliver exceptional user experiences.
* Use HTML, CSS, and JavaScript to create user-friendly interfaces and efficient, reliable code.
* Collaborate with product and design teams to clearly define requirements and devise solutions.
* Partner with backend developers to integrate systems and elevate product quality.
* Write, test, and maintain high-performance code compatible across various browsers and devices.
* Analyze page performance and suggest improvements for optimization.
* Participate in code reviews to improve code quality.

## Requirements:
* Bachelor's degree in Computer Science or a related field.
* 3-5 years of frontend development

## Experience.
* Proficiency in at least one modern JavaScript framework or library (Vue preferred).
* Experience with Git or similar version control systems.
* Strong understanding of front-end engineering tools, including Webpack and Babel.
* Basic knowledge of databases and backend development processes.
* Strong analytical and problem-solving skills with effective communication abilities.

## Preferred skills:
* Experience with a backend programming language.
* Experience with Test-Driven Development(TDD) or Behavior-Driven Development(BDD).
* Familiarity with DevOps and Cl/CD practices.
